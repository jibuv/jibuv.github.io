Formatting Tags¶

This is **bold**, *italic*, ***bold and italic***  
and this is ~~striked out~~

This can be written using underscores as well  
__bold__, _italic_ and  ___bold and italic___.

Use backslash to escape any markdown sensitive character.   
This is \*\*not bold\*\*.

This is bold, italic, bold and italic and this is ~~striked out~~

This can be written using underscores as well bold__, italic_ and ___bold and italic.

Use backslash to escape any markdown sensitive character.
This is **not bold**.

Blockquote is made by placing > in the beggining of the line, and >> for nested block quote

This is a block quote
that can span muliple lines

This is a nested block quote
that spanned into a second line

Line Breaks and Horizontal Lines¶

Line breaks (new line) can be made by placing two or more trailing spaces
at the end of the line.

For paragraph breaks, place a blank line between paragraphs.

Horizontal lines can be made by using three dashes ---, three underscores ___ or three asterisks ***

Code¶

Use a single ` to enclose inline code` eg; print("Hello")

Enclose multi line code with ``` in beginning and end. 

#To specify code as python ```python on beginning of code block
print("This is a python code block")

Links¶

To get a link in place use enclose in <>  
like this. <https://github.com/>

[Github](https://github.com 
"Optional title - click here to access github goes here")

To get a link in place use enclose in <> like this. https://github.com/

Github

Images¶

Image syntax is similar to that of a link, but with an exclamation in the beginning. 

![Alternate text placed here](/images/logo_200x200.png 
"Optional title placed here")

[![My github blog](/images/logo_200x200.png 
"Optional title - click here to access my github blog")]
(https://jibuv.github.io)

Use | symbol to seperate columns in a row 
and use 3 or more hyphen to seperate the 
column header from the data rows.

| 66  | 223  | 12    |
